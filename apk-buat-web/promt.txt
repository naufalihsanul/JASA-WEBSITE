# 🎨 Panduan Menulis Prompt Website Premium (Anti AI-Look)

## Kenapa Website dari AI Terlihat "AI-Generated"?

Sebelum ke solusi, pahami dulu masalahnya. Website yang terlihat AI biasanya punya ciri:

| ❌ Ciri AI-Look | ✅ Ciri Professional |
|---|---|
| Warna generik (biru polos, gradien ungu-biru) | Palet warna yang *curated* dan unik |
| Layout simetris sempurna, grid kaku | Asimetris intentional, whitespace berani |
| Font default atau 1 font saja | Typography pairing 2-3 font dengan hierarchy jelas |
| Hero section generik dengan heading besar + CTA | Hero section dengan komposisi editorial |
| Rounded corners berlebihan di mana-mana | Mix sharp & rounded sesuai konteks |
| Gradien yang sama di semua section | Gradien subtle, dipakai strategis |
| Shadow yang flat dan uniform | Shadow berlapis (layered shadows) |
| Animasi bounce/fade generic | Micro-interaction yang purposeful |
| Spacing tidak konsisten | Spacing system (4px/8px grid) |
| Border-radius seragam semua elemen | Variasi radius sesuai hierarki |

## 8 Aspek yang HARUS Ada di Prompt Kamu

### 1. 🎯 Design System & Framework CSS

Jangan hanya bilang "buat website bagus". Sebutkan **library/framework spesifik** yang menghasilkan output berkualitas:

```
Rekomendasi Library (pilih salah satu atau kombinasi):
```

| Library | Kegunaan | Kenapa Bagus |
|---|---|---|
| **Preline UI** | Component library | Komponen sudah polished, dark mode ready |
| **Aceternity UI** | Animated components | Efek-efek modern (spotlight, meteors, aurora) |
| **shadcn/ui** | Headless components | Clean, minimalis, sangat customizable |
| **GSAP** | Animation library | Animasi scroll-based profesional |
| **Locomotive Scroll** | Smooth scrolling | Efek parallax premium |
| **Lenis** | Smooth scroll | Alternatif Locomotive yang lebih ringan |
| **AOS (Animate On Scroll)** | Scroll animation | Mudah dipakai, efek reveal elegan |

> [!TIP]
> **Kombinasi terbaik untuk 1 file HTML**: Preline UI + AOS + Swiper.js. Ini sudah cukup untuk hasil yang sangat premium tanpa perlu build tools.

### 2. ✍️ Typography (SANGAT PENTING!)

Typography adalah **50% dari kesan profesional**. Kamu sudah pakai Jost + Radley, itu bagus! Tapi ada aturannya:

#### Rumus Typography Pairing:
```
Font Heading (Display/Serif) + Font Body (Sans-serif) + Font Accent (opsional)
```

#### Rekomendasi Pairing Premium:

| Heading | Body | Vibe |
|---|---|---|
| **Playfair Display** | **Inter** | Elegant, editorial, luxury |
| **Clash Display** (Fontshare) | **Satoshi** (Fontshare) | Modern, startup, tech |
| **Fraunces** | **Outfit** | Warm, friendly, premium |
| **Sora** | **DM Sans** | Clean, modern, SaaS |
| **Cabinet Grotesk** (Fontshare) | **General Sans** (Fontshare) | Bold, agency, creative |
| **Radley** | **Jost** | Classic, professional (punya kamu!) |
| **Space Grotesk** | **Inter** | Tech, futuristic |
| **Bricolage Grotesque** | **Geist** | Trendy 2024-2026, startup |

> [!IMPORTANT]
> **Di prompt, sebutkan SPESIFIK:**
> - Font heading: `[nama font]`
> - Font body: `[nama font]`
> - Font weight yang dipakai: misalnya heading `700-900`, body `400-500`
> - Letter-spacing: heading `-0.02em` (tighter), body `normal`
> - Line-height: heading `1.1-1.2`, body `1.6-1.8`

#### Contoh di prompt:
```
"Gunakan font Clash Display (weight 600-700) untuk heading dengan letter-spacing -0.03em 
dan line-height 1.1, serta Satoshi (weight 400-500) untuk body text dengan line-height 1.7"
```

### 4. 📐 Layout & Spacing

Ini yang paling sering bikin website terlihat AI: **spacing yang tidak intentional**.

#### Sebutkan di prompt:
```
"Gunakan spacing system kelipatan 8px. Section padding 120px vertikal. 
Max-width konten 1200px. Gunakan CSS Grid untuk layout utama. 
Beri whitespace yang luas dan berani — jangan takut kosong. 
Gunakan asymmetric grid (misalnya 5:7 atau 4:8 kolom) untuk section fitur, 
bukan grid simetris 6:6."
```

#### Detail spesifik yang bisa disebut:

| Aspek | Nilai Recommended | Catatan |
|---|---|---|
| Section padding | `100px - 160px` (vertical) | Semakin besar = semakin premium |
| Container max-width | `1200px - 1400px` | Jangan full-width tanpa constraint |
| Gap antar elemen | `24px - 48px` | Konsisten! |
| Heading margin-bottom | `24px - 32px` | Jarak heading ke paragraf |
| Card padding | `32px - 48px` | Card yang lapang terlihat mahal |
| Border-radius cards | `16px - 24px` | Untuk cards/containers |
| Border-radius buttons | `8px - 12px` atau `999px` (pill) | Pilih satu style, konsisten |

### 5. ✨ Animasi & Micro-Interactions

#### Sebutkan di prompt:
```
"Tambahkan animasi scroll reveal menggunakan AOS library dengan delay bertahap 
(staggered 100ms). Hover effect pada card: subtle scale(1.02) dengan shadow yang 
membesar. Transisi halus 0.3s cubic-bezier(0.4, 0, 0.2, 1) untuk semua interactive 
elements. Navbar blur backdrop-filter saat scroll. Smooth scroll behavior."
```

#### Animasi yang bikin premium:

| Efek | Implementasi | Kesan |
|---|---|---|
| **Staggered reveal** | AOS dengan delay bertahap | Konten muncul satu-satu, terasa crafted |
| **Parallax subtle** | `transform: translateY()` on scroll | Kedalaman visual |
| **Cursor follower** | Custom cursor dengan delay | Interaktif, modern |
| **Magnetic buttons** | Tombol yang sedikit "tertarik" ke cursor | Playful, premium |
| **Text reveal** | Karakter/kata muncul satu-satu | Editorial, high-end |
| **Blur navbar** | `backdrop-filter: blur(12px)` + transparency | Modern, glassmorphism |
| **Gradient animation** | Background gradient yang bergerak pelan | Hidup, dinamis |
| **Hover glow** | Box-shadow yang bercahaya saat hover | Futuristic |

> [!WARNING]
> Jangan overdo animasi! Pilih **3-5 jenis animasi** saja. Terlalu banyak animasi justru terlihat amatir.

### 6. 🧩 Komponen & Detail Visual

#### Sebutkan detail ini di prompt:

```
"Untuk cards: gunakan border 1px solid rgba(255,255,255,0.06), background 
semi-transparent, dan layered shadow. Untuk badges/tags: gunakan pill shape 
dengan background accent/10 dan text accent. Untuk divider: gunakan gradient 
line yang fade di kedua ujung, bukan garis solid. Untuk section background: 
tambahkan subtle grain texture overlay dan radial gradient blob."
```

#### Detail yang membedakan pro vs AI:

| Detail | AI-Look ❌ | Pro-Look ✅ |
|---|---|---|
| Card border | `border: 1px solid #333` | `border: 1px solid rgba(255,255,255,0.06)` |
| Shadow | `box-shadow: 0 4px 6px rgba(0,0,0,0.1)` | Shadow berlapis 2-3 layer |
| Background texture | Flat solid color | Grain overlay + gradient blobs |
| Section divider | `<hr>` atau garis lurus | Gradient line fade atau wave SVG |
| Button hover | Hanya ganti warna | Scale + shadow + glow |
| Image treatment | Gambar kotak biasa | Rounded + border + subtle shadow |
| Gradient usage | Gradient sebagai background utama | Gradient sebagai accent, subtle |

#### Contoh Layered Shadow (premium look):
```css
box-shadow: 
  0 1px 2px rgba(0,0,0,0.05),
  0 4px 8px rgba(0,0,0,0.04),
  0 16px 32px rgba(0,0,0,0.03);
```

### 7. 📱 Responsiveness & Performance

```
"Website harus fully responsive dengan breakpoint: mobile (< 768px), tablet 
(768px - 1024px), desktop (> 1024px). Gunakan clamp() untuk fluid typography. 
Lazy load semua gambar. Optimasi untuk Core Web Vitals."
```

#### Fluid Typography contoh:
```css
/* Bukan ini: */
h1 { font-size: 64px; }

/* Tapi ini: */
h1 { font-size: clamp(2.5rem, 5vw + 1rem, 4.5rem); }
```

## 💡 Pro Tips Tambahan

1. **Sebut referensi website nyata**: "Dengan visual style mirip Linear.app / Vercel.com / Stripe.com" — ini memberi AI konteks yang sangat kuat tentang level kualitas yang kamu mau.

2. **Gunakan font dari Fontshare**: Font dari [Fontshare](https://www.fontshare.com/) seperti Clash Display, Cabinet Grotesk, Satoshi, General Sans terlihat jauh lebih unik dibanding Google Fonts yang sudah terlalu common.

3. **Sebut "bento grid"**: Layout bento (seperti grid iOS widgets) membuat section fitur terlihat jauh lebih modern dibanding grid biasa.

4. **Minta grain texture**: `background-image: url("data:image/svg+xml,...")` atau CSS noise overlay memberi depth yang subtle.

5. **Sebut easing function spesifik**: `cubic-bezier(0.4, 0, 0.2, 1)` terlihat jauh lebih natural dibanding `ease` atau `linear`.

6. **Tambahkan "jangan generic"**: Kadang perlu disebut eksplisit agar AI tidak default ke pattern umum.

7. **Minta warna dengan opacity**: `rgba(255,255,255,0.06)` untuk border dan `rgba(accent,0.1)` untuk badge background lebih halus dibanding warna solid.
